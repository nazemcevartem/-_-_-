Инструкция по использованию:
1) Файл НТО_ИИ_Личный.ipynb нужно запустить в колаб, предварительно загрузив туда архив с данными
2) Если возникла проблема с bert_embeddings, то можно взять эмбеддинги из файла bert_embeddings.pkl
3) Рекомендуется запускать на gpu, а то на cpu будет около часа выполняться
4) Библиотеки импортируются сразу, без предварительной установки, но вот версии
