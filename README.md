Инструкция по использованию:
1) Файл НТО_ИИ_Личный.ipynb нужно запустить в колаб, предварительно загрузив туда архив с данными
2) Если возникла проблема с bert_embeddings или они скачиваются очень долго, то можно взять эмбеддинги из файла bert_embeddings.pkl, загрузить его в колаб и оттуда возмутся эмбеддинги - на него прикреплена ссылка в текстовом файле files.docx
3) Рекомендуется запускать на gpu, а то на cpu будет долго выполняться
4) Библиотеки импортируются сразу, без предварительной установки, но вот версии
   numpy -  2.0.2
   pandas - 2.2.2
   joblib - 1.5.2
   lightgbm - 4.6.0
   torch - 2.8.0+cu126
   sklearn - 1.6.1
   transformers - 4.57.1
5) На всякий случай ссылка на файл processed_features.parquet тоже прикреплена в файле, файл feature_names.pkl и lgb_model.txt прикреплены в сам проект в корень
6) Файл AI_USAGE.md - в корне проекта
7) Метрика в лидерборде с этого решения: 0,7658219433179752
